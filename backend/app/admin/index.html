MCP integrations

MCP runs server-side — the phone never speaks MCP. This keeps Plane/Notion credentials off the device and lets the dispatch worker own idempotency and retries.

🔒Tokens live in Azure Key Vault. Store the secret name in auth_ref — never paste the token itself.
